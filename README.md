## 🌐 Live Demo

**https://khanblair.github.io/hotel-website/**

## ✨ Features

- 🌓 **Dark/Light Theme Toggle** - Persists user preference in localStorage
- 📱 **Fully Responsive** - Optimized for mobile, tablet, and desktop
- 🎠 **Hero Image Slider** - Auto-play with navigation controls
- 🔍 **Room Filtering** - Filter rooms by category (Standard, Deluxe, Suite, Family)
- 📋 **FAQ Accordion** - Smooth expand/collapse animations
- 📧 **Contact Form** - Full validation with success feedback
- 🧮 **Animated Statistics** - Scroll-triggered counter animations
- 📜 **Scroll Animations** - Elements animate on scroll into view
- 🍔 **Mobile Menu** - Hamburger menu for mobile devices
- ⬆️ **Sticky Navigation** - Header stays fixed on scroll

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid, animations
- **Vanilla JavaScript** - No frameworks, pure JS
- **Font Awesome 6.4** - Icons
- **Google Fonts** - Playfair Display & Poppins
- **Unsplash** - High-quality placeholder images

## 🚀 Getting Started

### Local Development

```bash
# Clone the repository
git clone https://github.com/khanblair/hotel-website.git

# Navigate to project
cd hotel-website

# Start a local server (Python)
python3 -m http.server 8000

# Or use any static server
# npx serve .
# or
# php -S localhost:8000
```

Then open **http://localhost:8000** in your browser.
